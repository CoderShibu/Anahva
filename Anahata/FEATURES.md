## ðŸ‡®ðŸ‡³ India-Specific Features

### A. Confidential Mode (Trust Multiplier)
- **Location**: Home page toggle
- **Features**:
  - One-tap mode where no data is saved
  - No memory is stored
  - No history is shown
  - Journals auto-delete on exit
  - AI memory disabled temporarily
- **Why**: Addresses fear of family finding thoughts, phone checks, and social stigma

### B. Regional Emotional Context Engine
- **Location**: Chat page
- **Features**:
  - AI understands Indian emotional realities:
    - Family pressure ("parents expect", "relatives keep asking")
    - Academic rank anxiety
    - Comparison culture
    - Career guilt
  - Culturally sensitive responses
  - No Westernized advice
- **Detection**: Automatically detects Indian emotional phrases and responds appropriately

### C. Night-Watch Mode
- **Location**: Automatically activates after midnight (12 AM - 6 AM)
- **Features**:
  - Special UI with softer gold glow
  - Slower UI animations
  - AI defaults to gentle listening mode
  - Fewer prompts
  - Longer response times for calm
  - Visual indicator in Chat and Home
- **Why**: Most breakdowns in India happen late night, in silence, alone

### D. Safe-Note to Self
- **Location**: Safe Circle page
- **Features**:
  - User writes a note to their future self
  - Shown later during low-mood periods
  - Fully private, user-controlled
  - Respects Confidential Mode
- **Why**: Builds self-trust and emotional continuity

### E. Parent/Friend Safe-Signal
- **Location**: Safe Circle page
- **Features**:
  - Pre-written message templates
  - One-tap send via WhatsApp/Share API
  - Default: "I'm not okay, can we talk?"
  - Can add/edit custom messages
  - No explanation needed
- **Why**: People struggle to ask for help directly

### F. Exam & Work Stress Mode
- **Location**: Safe Circle page
- **Features**:
  - Temporary mode for:
    - Exams
    - Work pressure
    - Internships
  - AI reduces emotional probing
  - Focus on grounding + reassurance
  - No long reflections
  - Visual indicator when active
- **Why**: Students + young professionals are the largest base

### G. Trust Badges
- **Location**: Home page (bottom)
- **Features**:
  - Visual indicators:
    - "Encrypted on Device"
    - "No Data Selling"
    - "No Ads"
    - "Anonymous by Default"
- **Why**: Trust must be seen, not explained

# Anahva - Mental Wellness Sanctuary

A beautiful, modern mental wellness application built with React, TypeScript, and Tailwind CSS. Anahva provides a sacred space for journaling, emotional support, and mental health tracking.

## Features

- 🌙 **Dark & Light Themes** - Toggle between beautiful dark and light themes
- 🌍 **Multi-language Support** - English and Hindi (हिंदी)
- 📝 **AI-Powered Journaling** - Write from the heart with guided prompts
- 💬 **Empathetic Chat Companion** - Talk to Anahva for emotional support
- 📊 **Insights & Analytics** - Track your emotional journey over time
- 🛡️ **Safe Circle** - Access grounding exercises and support resources
- 🔒 **Privacy First** - All data encrypted on your device

## Quick Start

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Navigate to the project directory:
```bash
cd Anahva
```

2. Install dependencies:
```bash
npm install
```

### Running the Development Server

Start the development server:
```bash
npm run dev
```

The application will be available at:
- **Local:** http://localhost:5173
- **Network:** The terminal will show the network URL

### Building for Production

Create a production build:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Key Technologies

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server

- **React Router** - Navigation
- **TanStack Query** - Data fetching
- **Radix UI** - Accessible components

## Theme System

The app supports both dark and light themes:
- Toggle theme from the Navigation sidebar (desktop) or Settings page
- Theme preference is saved to localStorage
- Smooth transitions between themes

## Language Support

Currently supports:
- English (EN)
- Hindi (हिंदी)

Toggle language from the top-right corner on login or from Settings.

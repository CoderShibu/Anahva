## 🔐 Authentication Flow

### Step 1: Login

**Option A: Demo Login (Development)**
```javascript
const response = await fetch('http://localhost:3000/api/auth/demo', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    name: 'Shibasish',
    password: 'Shibasish'
  })
});

const { token, session } = await response.json();
console.log(token); // JWT token for API calls
```

**Option B: Anonymous Session (Production)**
```javascript
const response = await fetch('http://localhost:3000/api/auth/anonymous', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    language: 'EN' // or 'HI'
  })
});

const { token, session } = await response.json();
```

### Step 2: Store Token

```javascript
// Store in memory or sessionStorage (NOT localStorage for security)
sessionStorage.setItem('anahva_token', token);
```

### Step 3: Use Token in All Protected Calls

```javascript
const token = sessionStorage.getItem('anahva_token');

const response = await fetch('http://localhost:3000/api/journal/list', {
  headers: {
    'Authorization': `Bearer ${token}`
  }
});
```

## 📔 Journal Integration

### Create Journal Entry

**IMPORTANT: Encrypt on frontend before sending!**

```javascript
// 1. Get encryption key from somewhere (backend provides, or frontend generates)
const plaintext = "Today I'm feeling stressed...";
const encryptedPayload = await encryptContent(plaintext); // Your encryption

// 2. Send to backend
const token = sessionStorage.getItem('anahva_token');

const response = await fetch('http://localhost:3000/api/journal/create', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    encrypted_payload: encryptedPayload, // Base64 encoded
    allow_ai_memory: false // User controls AI memory
  })
});

const { journal } = await response.json();
console.log(journal.id); // Save for later deletion
```

### Fetch Journal Entries

**IMPORTANT: Backend returns encrypted data!**

```javascript
const token = sessionStorage.getItem('anahva_token');

const response = await fetch(
  'http://localhost:3000/api/journal/list?limit=50&offset=0',
  {
    headers: {
      'Authorization': `Bearer ${token}`
    }
  }
);

const { journals, pagination } = await response.json();

// journals[0].encrypted_payload is BASE64 ENCRYPTED DATA
for (const journal of journals) {
  // Decrypt on frontend
  const plaintext = await decryptContent(journal.encrypted_payload);
  console.log(plaintext); // Original journal content
}
```

### Delete Journal Entry

```javascript
const token = sessionStorage.getItem('anahva_token');
const journalId = 'uuid-from-create-response';

const response = await fetch(
  `http://localhost:3000/api/journal/${journalId}`,
  {
    method: 'DELETE',
    headers: {
      'Authorization': `Bearer ${token}`
    }
  }
);

const { success } = await response.json();
```

## 💬 Chat / AI Integration

### Send Message and Get AI Response

**AI response is generated by Gemini API in the backend!**

```javascript
const token = sessionStorage.getItem('anahva_token');
const userMessage = "I'm feeling overwhelmed";

const response = await fetch('http://localhost:3000/api/chat/message', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    message: userMessage,
    mode: 'CALM', // 'LISTEN', 'REFLECT', or 'CALM'
    allow_memory: false // User controls AI memory
  })
});

const { response: aiMessage, mode, language } = await response.json();
console.log(aiMessage); // "Take a deep breath. You're safe here..."
```

### Chat Modes

| Mode | Best For | AI Behavior |
|------|----------|------------|
| `LISTEN` | General conversation | Empathetic listening, open questions |
| `REFLECT` | Self-discovery | Socratic questioning, guided reflection |
| `CALM` | Stress/anxiety | Grounding, breathing, calming techniques |

## 🔄 Complete Example: Full Journal Flow

```javascript
// ===== FRONTEND INTEGRATION EXAMPLE =====

class AnahvaAPI {
  constructor() {
    this.baseURL = 'http://localhost:3000';
    this.token = null;
  }

  // ✅ Step 1: Login
  async login() {
    const response = await fetch(`${this.baseURL}/api/auth/demo`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        name: 'Shibasish',
        password: 'Shibasish'
      })
    });

    const { token } = await response.json();
    this.token = token;
    sessionStorage.setItem('anahva_token', token);
    return token;
  }

  // ✅ Step 2: Create Journal Entry
  async createJournal(plaintext, allowAIMemory = false) {
    // Frontend encrypts
    const encryptedPayload = await this.encryptContent(plaintext);

    const response = await fetch(`${this.baseURL}/api/journal/create`, {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${this.token}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        encrypted_payload: encryptedPayload,
        allow_ai_memory: allowAIMemory
      })
    });

    const { journal } = await response.json();
    return journal;
  }

  // ✅ Step 3: Get All Journals
  async getJournals() {
    const response = await fetch(`${this.baseURL}/api/journal/list`, {
      headers: { 'Authorization': `Bearer ${this.token}` }
    });

    const { journals } = await response.json();

    // Frontend decrypts each journal
    const decrypted = [];
    for (const j of journals) {
      const plaintext = await this.decryptContent(j.encrypted_payload);
      decrypted.push({
        ...j,
        plaintext
      });
    }

    return decrypted;
  }

  // ✅ Step 4: Chat with AI
  async chat(message, mode = 'LISTEN') {
    const response = await fetch(`${this.baseURL}/api/chat/message`, {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${this.token}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        message,
        mode,
        allow_memory: false
      })
    });

    const { response: aiMessage } = await response.json();
    return aiMessage;
  }

  // ✅ Step 5: Delete Journal
  async deleteJournal(journalId) {
    const response = await fetch(
      `${this.baseURL}/api/journal/${journalId}`,
      {
        method: 'DELETE',
        headers: { 'Authorization': `Bearer ${this.token}` }
      }
    );

    return await response.json();
  }

  // Your encryption/decryption methods
  async encryptContent(plaintext) {
    // Implement frontend encryption here
    return btoa(plaintext); // Placeholder
  }

  async decryptContent(encryptedPayload) {
    // Implement frontend decryption here
    return atob(encryptedPayload); // Placeholder
  }
}

// ===== USAGE EXAMPLE =====

const api = new AnahvaAPI();

async function main() {
  // 1. Login
  await api.login();
  console.log('✅ Logged in');

  // 2. Create journal
  const journal = await api.createJournal('I felt anxious today');
  console.log('✅ Journal created:', journal.id);

  // 3. Get all journals
  const journals = await api.getJournals();
  console.log('✅ Retrieved', journals.length, 'journals');

  // 4. Chat with AI
  const aiResponse = await api.chat('I am stressed', 'CALM');
  console.log('✅ AI said:', aiResponse);

  // 5. Delete journal
  await api.deleteJournal(journal.id);
  console.log('✅ Journal deleted');
}

main().catch(console.error);
```

## 🚨 Error Handling

### Common Error Responses

**401 Unauthorized - Invalid Token**
```javascript
if (response.status === 401) {
  // Token expired or invalid
  // Redirect to login
  await api.login();
}
```

**400 Bad Request - Invalid Input**
```javascript
if (response.status === 400) {
  const { error, message } = await response.json();
  console.error(`${error}: ${message}`);
  // Show user error message
}
```

**429 Too Many Requests - Rate Limited**
```javascript
if (response.status === 429) {
  // Wait and retry
  await new Promise(resolve => setTimeout(resolve, 1000));
  // Retry request
}
```

**500 Internal Server Error**
```javascript
if (response.status === 500) {
  // Backend is down or errored
  console.error('Backend error');
  // Show user: "Service temporarily unavailable"
}
```

### Complete Error Wrapper

```javascript
async function callAPI(method, endpoint, body = null) {
  const token = sessionStorage.getItem('anahva_token');
  
  try {
    const response = await fetch(`http://localhost:3000${endpoint}`, {
      method,
      headers: {
        'Authorization': `Bearer ${token}`,
        'Content-Type': 'application/json'
      },
      body: body ? JSON.stringify(body) : null
    });

    if (!response.ok) {
      const error = await response.json();
      throw new Error(
        `${response.status}: ${error.error} - ${error.message}`
      );
    }

    return await response.json();
  } catch (error) {
    console.error('API Error:', error);
    // Show user-friendly error
    throw error;
  }
}
```

## 📱 Encryption on Frontend

Since backend stores encrypted data, frontend must encrypt/decrypt:

### Example with crypto-js (simple)

```javascript
import CryptoJS from 'crypto-js';

const FRONTEND_SECRET = 'your-secret-key';

function encryptContent(plaintext) {
  return CryptoJS.AES.encrypt(plaintext, FRONTEND_SECRET).toString();
}

function decryptContent(encryptedData) {
  const bytes = CryptoJS.AES.decrypt(encryptedData, FRONTEND_SECRET);
  return bytes.toString(CryptoJS.enc.Utf8);
}
```

### Example with libsodium (production)

```javascript
import sodium from 'libsodium.js';

const key = sodium.crypto_box_keypair();

async function encryptContent(plaintext) {
  const nonce = sodium.randombytes_buf(
    sodium.crypto_box_NONCEBYTES
  );
  const encrypted = sodium.crypto_box_easy(
    plaintext,
    nonce,
    key.publicKey,
    key.privateKey
  );
  return sodium.to_base64(encrypted);
}
```
